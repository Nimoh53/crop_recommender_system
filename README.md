# Crop Recommender System

## Description

This project recommends suitable crops based on soil and environmental conditions.

## Dataset Setup

This project uses a large dataset that is not included in the repository due to GitHub file size limits.

### Download the dataset

https://drive.google.com/file/d/1ZVpbDPy9lxxBSZhc7oZyIGZB-d_SZPq3/view?usp=sharing

### Setup Instructions

1. Download the dataset
2. Place it in the `data/` folder
3. Rename it to `soil_data.csv`

### Notes

* A sample dataset (`sample_soil_data.csv`) is included for testing
* The full dataset is required for accurate results

## How to Run

1. Install dependencies
2. Run the application
